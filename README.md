## ğŸš€ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) (v14 or higher) and npm installed.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Anil-sai0418/online-notes.git
   cd online-notes
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` (or the port shown in your terminal) to view the app.

## ğŸ“– Usage

- **Creating a Note**: Click the "+" button or press `Cmd/Ctrl + Enter` to create a new note.
- **Editing**: Just start typing! Use the toolbar for formatting.
- **Adding Images**: Paste images directly or use the upload feature.
- **Voice Typing**: Click the microphone icon in the toolbar to start dictation.
- **Locking a Note**: Click the lock icon in the toolbar, set a password, and your note content will be hidden and encrypted.
- **Deleting**: Select a note and press `Ctrl + Delete/Backspace` or use the delete button.
